# 第六周 循环

写出下列三个循环的输出值，并解释原因

```js
for (var i = 0; i < 10; i++ ) {
	setTimeout(() => {
		console.log(i)
	})
}

for (var i = 0; i < 10; i++ ) {
	(function (i) {
		setTimeout(() => {
			console.log(i)
		})
	}) (i)
}

for (let i = 0; i < 10; i++ ) {
	setTimeout(() => {
		console.log(i)
	})
}
```
