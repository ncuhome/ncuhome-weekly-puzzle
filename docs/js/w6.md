# 第六周：闭包

观察以下JS代码，并判断会输出什么

```javascript
function foo(m, n) {
  console.log(n)
  return {
    foo: function(o) {
      return foo(o, m)
    }
  }
}

var a = foo(0);
a.foo(1);
a.foo(2);

var b = foo(0).foo(1);
b.foo(1);
b.foo(2);
```
